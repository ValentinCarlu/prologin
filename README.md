# Prologin

Mes solutions aux exercices du concours **Prologin**, réalisés pour m'entraîner pour ce concours afin d'aller le plus loin possible dans la compétition.

## Langage
Python

## Organisation du repo
- Le dossier `Entrainement Qualification 2024` contient des codes que j'ai écrits pour découvrir le concours Prologin. [Problèmes ici](https://prologin.org/train/2024/qualification)
- Le dossier `Qualification 2025` contient les codes qui m'ont permis de me qualifier pour les épreuves régionales de 2025. [Problèmes ici](https://prologin.org/train/2025/qualification)
- Le dossier `Entrainement ER 2024` contient les codes que j'ai écrits dans le but de m'entraîner aux épreuves régionales de 2025. Les problèmes proviennent des épreuves régionales de 2024. [Problèmes ici](https://prologin.org/train/2024/semifinal)

## Commentaires

Je participe également à l'édition 2026 de Prologin. Les codes de cette année seront ajoutés ici une fois la phase de qualification terminée.
